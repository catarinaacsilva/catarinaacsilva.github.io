# Catarina Silva - Academic Portfolio

Welcome to my personal academic website and teaching repository. This project serves as a centralized platform to showcase my research activities, publications, and course materials.

## Overview

This website is hosted using **GitHub Pages** and is designed to provide an elegant, professional interface for students, colleagues, and researchers.

### Key Features
*   **Biography & Academic Path**: Details on my background and academic achievements.
*   **Research Interests**: Information on my work in Cybersecurity, Privacy Engineering, and DevPrivOps.
*   **Project Showcase**: Highlights of current and past research initiatives (e.g., RIGOUROUS, NEXUS).
*   **Publications**: Links to my research work via ORCID, Google Scholar, and ResearchGate.
*   **Teaching**: Interactive access to course materials, slides, and practical guides for students at ESTGA (University of Aveiro).

## Structure

*   `/ucs/`: Contains the specific pages for each course taught.
*   `/materiais/`: Repository for all downloadable and viewable academic documents (slides, guides, and supporting materials).
*   `/img/`: Image assets used across the website.
*   `index.html`: Main landing page of the portfolio.
*   `styles.css`: CSS stylesheet ensuring a clean, professional aesthetic.
*   `script.js`: Logic for managing interactive content and document viewing.

## Teaching Materials

The teaching section is organized by course and lecture. To access study materials:
1. Navigate to the **Teaching** section on the main page.
2. Select the desired course.
3. Choose the specific lecture to view or export the corresponding slides and guides.
